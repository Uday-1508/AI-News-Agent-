# 📰 AI News Agent

An AI-powered news assistant built using **CrewAI**, **Streamlit**, and **Serper**. It searches the web for the latest AI news and generates a well-structured summary using an LLM.

## 🚀 Features

* Search the latest AI news
* AI-generated news summaries
* Streamlit web interface
* CrewAI agent workflow
* Google search with Serper API

## 🛠️ Tech Stack

* Python
* CrewAI
* Streamlit
* Serper API
* Gemini API

## ⚙️ Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📌 Future Improvements

* Multiple AI agents
